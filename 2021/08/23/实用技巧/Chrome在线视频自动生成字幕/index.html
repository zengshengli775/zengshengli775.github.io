# 步骤

安装 Chrome 浏览器正式版最新版本，在地址栏输入：

chrome://flags/#enable-accessibility-live-captions

然后把 Default 选项更改为 Enabled。

根据提示重启浏览器。

再在浏览器的设置页面，点击高级 - 无障碍。

接着勾选开启为您的媒体生成字幕这个选项。接着勾选开启为您的媒体生成字幕这个选项。

设置好以后，当你再次在网页里播放视频时，视频下方将会出现一个新的灰色工具栏，生成的字幕将会不断出现在灰色工具栏中。
